# 塩の道

『塩の道』は、内陸の村と沿岸部の街を結ぶ街道を舞台にした、同時プロット型のチキンレース＆サバイバル・ボードゲームです。プレイヤーは商会となり、特産品を運んで塩（勝利点）と交換し、無事に村まで帰還することを目指します。

ブラウザ上で動作し、AIとのソロプレイや、PeerJSを用いたP2Pのオンライン対戦（2〜5人）が可能です。

[https://yasusho.github.io/salt_game/](https://yasusho.github.io/salt_game/)

## ゲームの目的

1. **往路（チキンレース）**: 特産品カードをめくり、荷崩れ（バースト）しないようにギリギリまで手札（資源）を集める。
2. **中間（塩の取引）**: 集めた特産品を支払って、沿岸部で塩（勝利点）を購入する。
3. **復路（サバイバルレース）**: 手札のカードを「旅の糧（移動力）」として消費しながら、罠を潜り抜けて誰よりも早く村へ生還する。

最終的に**生還したプレイヤーの中で、最も塩の点数が高いプレイヤー**が勝利となります。道中で手札（移動力）が尽きると「行き倒れ」となり脱落してしまいます。

## 遊び方

1. `index.html` をブラウザで開きます。
2. **ソロプレイの場合**: 参加人数（2〜5人）を選択し、「ソロプレイ開始」をクリックします。
3. **オンライン対戦の場合**: 
   - **ホスト**: 参加人数を選択し、「P2P部屋作成」をクリックして発行されたURL（または部屋ID）を相手に共有します。
   - **ゲスト**: 共有されたURLを開くか、「P2P部屋に参加」から部屋IDを入力して参加します。

詳しいルールについては、ゲーム内の「ルール」ボタン、または `rules.md` をご参照ください。
